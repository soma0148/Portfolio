02.02.01 EMNESITE


In the topic assignment, I was able to use the knowledge I gained from the first assignment to set up and design my own topic site. In order to start the task, I first had to come up with a topic that I am interested in. There I used the "mind map" method, where I wrote down my thoughts and got different ideas. I have an understanding of Yoga. At that moment I could imagine how I could make the site - the style, the colors, the composition.
To set up the pages, I made wireframes. For the cover, I have chosen to create an asymmetrical composition, the images partly overlap each other. To implement the idea, I have made a layout diagram, and have made several grids around the divers, because I can make the images responsive. During the task, I had changed it in the code to change the order in the mobile version.
